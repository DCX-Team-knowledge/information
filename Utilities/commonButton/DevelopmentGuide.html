Common Button LWC

The Common Button LWC is built as a template code implementing a configurable button component tailored for Salesforce applications.

Key Features:

  • Accessibility: The aria-label attribute is dynamically set using the assistiveText property, improving accessibility for screen readers.
  • Styling: The class attribute binds to buttonClasses, allowing dynamic assignment of CSS classes for styling purposes.
  • Interactivity: The disabled attribute is dynamically controlled through the disabled property, enabling or disabling the button as needed.
  • Content Slotting: The <slot></slot> element allows the insertion of child nodes, enabling flexible content placement inside the button.
